Untuk menjalankan : 
1. Ketik composer update
2. Ketik cp .env.example .env
3. Bikin database
4. Masukan nama database pada .env
5. Ketik php artisan key:generate
6. Ketik php artisan migrate
7. Ketik npm install
8. Ketik php artisan serve
9. Ketik npm run dev
